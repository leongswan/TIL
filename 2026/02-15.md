# TIL（2026/02/15）— Pythonのスライス（slice）

## 今日学んだこと（要点）
- **スライス**はPythonの特徴的な機能で、**文字列やリストから一部を抜き出す**ときに使う。
- 文字列も **リストのようにインデックスで扱える**ので、部分文字列を取り出せる。

---

## スライスの基本形
```python
some_list[start:stop:step]
```
## ルール
- stop（終点）に指定した位置は含まれない

- start / stop / step は すべて省略可能

- start / stop / step は 正負どちらも指定可能

- step を負にすると 逆順になる

- step が正のときは start < stop、step が負のときは start > stop を意識すると良い

---

## 例
```python
word = "Hello!Takashi."
word[1:4]  # "ell"（終点4は含まれない）
```

---

## 演習：スライスで指定部分を抜き取る
```python
word = "StayAsdfLy-aktjSCswkjfhuyhdudnfgerFy.fneuf-dxasdkf.hsilooF"

# 前から4文字
word_1 = word[:4]

# 26文字目から37文字目の偶数番目
# ※「26文字目」は1始まりなので、0始まりでは index 25
# ※ 26〜37文字目（両端含む）を取りたい場合、stopは 37 の次 → 37(1始まり) = index36 なので stop=37
# ※ 偶数番目（26,28,30,...）は index25 から 2ずつ
word_2 = word[25:37:2]

# 16文字目から10文字目を逆順で偶数番目
# 16文字目= index15, 10文字目= index9
# 16→10 を逆順に取るので step=-2
# stopは含まれないので、10文字目(index9)を含めるには stop=8 まで下げる
word_3 = word[15:8:-2]

# 後ろから8文字を逆順
word_4 = word[:-9:-1]

print(word_1 + word_2 + word_3 + word_4)
```

---

## 今日のまとめ
- スライスは start:stop:step で部分取得できる（stopは含まれない）

- 文字列もスライス可能（部分文字列を抜ける）

- step を負にすると逆順、-2 のようにすると「1つ飛ばしの逆順」もできる
