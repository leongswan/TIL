## 1. Pythonのファイル操作（open / read / write / close）
- Pythonでは外部ファイルを **開く→読む/書く→閉じる** という流れで扱う。
- よく使うのは `with open(...) as f:` 構文  
  - **自動でcloseしてくれる**ので安全・読みやすい（Pythonic）。

### 改行ごとに読み込んで出力する代表パターン
- 従来の書き方
  - `f = open(...)`
  - `for row in f: print(row)`
  - `f.close()`
- 推奨（with構文）
  - `with open(...) as f:`
  - `for row in f: print(row)`
```python
f = open('ファイル名')          with open('ファイル名')as f: 
for row in f:                  for row in f: 
  print(row)                     print(row) 
f.close
```

## 2. CSVファイルを1行ずつ処理する（fruits.csv）
### 目的
- `fruits.csv` を1行ずつ読み、カンマ区切りで分割して、
  `〇〇は△△円です。` の形式で出力する。
```python
with open('fruits.csv') as f: 
 for row in f: 
  col = row.split(",") 
  fruit = col[0] 
  price = str(col[1].rstrip()) 
  print(f"{fruit}は{price}円です。",end='')
 ``` 

### 学び（ポイント）
- `row.split(",")` でCSVの列を分割できる。
- `rstrip()` で末尾の改行（`\n`）を削除できる。
- `end=''` を指定すると **printの自動改行を抑制**できる。

## 3. クラスとオブジェクト指向（OOP）
### 重要用語
- **クラス**：設計図
- **インスタンス**：クラスから作られた実体（オブジェクト）
- **メソッド**：クラスに属する関数
- **インスタンス変数**：`self.xxx` の形で各インスタンスが持つ値
- **クラス変数**：クラス全体で共有する変数
- **__init__**：インスタンス生成時に自動で呼ばれる初期化メソッド

## 4. 継承（Inheritance）
- 既存クラス（親クラス）の機能を引き継いで、新しいクラス（子クラス）を作る仕組み。
- 子クラスの `__init__` では、親クラスの `__init__` を呼び出して初期化する。

### 例：Book → Bunko（文庫）
- `Bunko(Book)` のように書くと、`Bunko` は `Book` を継承する。
- `Book.__init__(self, title, price)` を呼ぶことで、親クラスの初期化を実行できる。
