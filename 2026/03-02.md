# TIL（2026/3/2）C言語：条件分岐 / break / continue / if と else if の違い

## 1) 条件分岐（if文）の役割
- 条件（真/偽）によって **実行する処理を分ける**ための構文
- `if` / `else if` / `else` を使うと、上から順に条件を評価して分岐できる

---

## 2) break文 / continue文（ループ制御）
### break
- **今動いているループをその場で終了**して、ループの外へ出る
- 例：ある条件が出たら探索をやめたいとき

### continue
- **今の周回だけをスキップ**して、次の周回へ進む
- 例：特定の値だけ処理したくないとき

---

## 3) for文での break / continue の例
※ `break` と `continue` は同時に実行されないので、通常は条件を分けて書く。

```c
#include <stdio.h>

int main(void){
    for (int i = 0; i < 100; i++){
        if (i == 16){
            break;  // 16になったらループ終了（以降は出力されない）
        }
        printf("%d\n", i);
    }
    return 0;
}
```

```c
#include <stdio.h>

int main(void){
    for (int i = 0; i < 100; i++){
        if (i == 16){
            continue;  // 16だけ飛ばして次の周回へ
        }
        printf("%d\n", i);
    }
    return 0;
}
```

## 4) 「if と else if」 vs 「if を2回」 の違い

ここが重要：**条件が両方成立する可能性があるとき、挙動が変わる。**

### (A) if ～ else if（どれか1つだけ実行）
- 上の条件が真なら、下の条件は評価されない（排他的）
```c
if (20 <= age){
    printf("adult\n");
}
else if (10 <= age){
    printf("teenager\n");
}
```
例：`age = 20` のとき
- `adult` だけ出力（`teenager` は出ない）

##

### (B) if を2回（両方実行され得る）
- それぞれ独立した条件判定なので、**両方真なら両方実行**
```c
if (20 <= age){
    printf("adult\n");
}
if (10 <= age){
    printf("teenager\n");
}
```
例：`age = 20` のとき
- `adult` と `teenager` の **両方が出力される**

---

## 5) 例（あなたのコード：age = 20）
```c
#include <stdio.h>

int main(void){
    int age = 20;

    if (20 <= age){
        printf("adult\n");
    }
    else if (10 <= age){
        printf("teenager\n");
    }

    return 0;
}
```

---

## 今日のまとめ

- `break`：ループを終了して外に出る
- `continue`：その周回だけ飛ばして次へ進む
- `if ～ else if`：条件は「どれか1つ」だけ実行（排他的）
- `if を2回`：条件が両方真なら「両方」実行される
