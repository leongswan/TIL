## 学んだ内容

### 1. 1年間継続利用するEC2の購入オプション
1年間継続利用が決まっているアプリケーション開発向けに、  
どのEC2購入オプションを選ぶべきかを学んだ。

#### 正解
- **リザーブドインスタンス（Reserved Instances）**

#### 学んだこと
- 1年または3年の継続利用が決まっている場合は、リザーブドインスタンスが有効
- オンデマンド料金より大きな割引を受けられる
- 長期利用前提のワークロードでは、コスト最適化に役立つ
- 一方で、短期利用や利用量が読めない場合はオンデマンドの方が向いていることもある

#### ポイント
- **継続利用が前提なら RI**
- **柔軟性重視ならオンデマンド**
- **中断可能な処理ならスポットインスタンス**

---

### 2. 複数AWSアカウントの一括請求
会社内で複数のAWSアカウントを利用している場合に、  
請求をまとめて管理するためのサービスを学んだ。

#### 正解
- **AWS Organizations**

#### 学んだこと
- AWS Organizations には一括請求機能がある
- 複数アカウントの請求と支払いを統合できる
- 組織単位でアカウント管理しやすくなる
- ガバナンスやポリシー管理にも役立つ

#### ポイント
- 複数アカウント運用では、Organizations が重要
- 一括請求により、請求管理の手間を減らせる
- 組織全体でのコスト把握がしやすくなる

---

### 3. コスト最適化の確認に使うサービス
現在のAWS利用状況がコスト最適化されているかを確認するために、  
利用できるサービスを2つ学んだ。

#### 正解
- **AWS Trusted Advisor**
- **AWS Cost Explorer**

#### 学んだこと
- Trusted Advisor は、コスト削減・可用性・パフォーマンス・セキュリティなどの観点でベストプラクティスチェックを行う
- Cost Explorer は、AWSの利用状況や支出を分析できる
- Cost Explorer の適切なサイズ設定に関する推奨事項では、EC2インスタンスの縮小や終了によるコスト削減機会を見つけられる
- コスト最適化では、ただ料金を見るだけでなく、**利用状況の分析** と **改善提案の確認** が重要

#### ポイント
- **Trusted Advisor**：改善点やベストプラクティスを確認する
- **Cost Explorer**：利用料金や使用傾向を分析する
- コスト削減には、未使用・過剰スペックなリソースの見直しが大切

## 今日の重要キーワード

| 用語 | 意味 |
|---|---|
| リザーブドインスタンス | 1年または3年の利用を前提に割引を受けられるEC2購入オプション |
| オンデマンドインスタンス | 必要なときに使い、使った分だけ課金される購入オプション |
| スポットインスタンス | 余剰キャパシティを低価格で利用できるが、中断される可能性があるオプション |
| AWS Organizations | 複数のAWSアカウントを一元管理できるサービス |
| 一括請求 | 複数アカウントの請求をまとめて管理する仕組み |
| AWS Trusted Advisor | ベストプラクティスに基づく改善提案を確認できるサービス |
| AWS Cost Explorer | AWS利用料金や使用状況を可視化・分析できるサービス |
| 適切なサイズ設定 | リソースの性能や規模を用途に合った適正サイズに見直すこと |
