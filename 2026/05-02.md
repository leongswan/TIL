## 学習テーマ

AWS インスタンスコネクタによる接続

## 今日学んだこと

AWSマネジメントコンソールから、EC2インスタンスへブラウザ上で接続する方法を学んだ。

今回は、MacのTerminalからSSH接続する方法ではなく、AWSコンソールの **EC2 Instance Connect** を使って接続する流れを確認した。

---

## EC2 Instance Connectとは

EC2 Instance Connectは、AWSマネジメントコンソール上からEC2インスタンスへ接続できる機能。

Terminalを開いてSSHコマンドを入力しなくても、ブラウザ上でLinuxサーバーを操作できる。

---

## 接続の流れ

### 1. EC2インスタンス一覧を開く

AWSマネジメントコンソールでEC2のインスタンス一覧画面を開く。

接続したいインスタンスを選択する。

確認するポイントは以下。

| 項目 | 内容 |
|---|---|
| インスタンス名 | 接続したいEC2を確認する |
| インスタンスID | 対象インスタンスを識別するID |
| インスタンスの状態 | `実行中` であることを確認する |
| ステータスチェック | 正常に起動しているか確認する |
| パブリックIPv4アドレス | 外部から接続するためのIPアドレス |

---

## 2. 「接続」ボタンを押す

接続したいインスタンスを選択した状態で、画面上部の **接続** ボタンをクリックする。

すると、インスタンスへの接続方法を選ぶ画面が表示される。

---

## 3. EC2 Instance Connectを選択する

接続方法の画面では、以下のようなタブが表示される。

| タブ | 内容 |
|---|---|
| EC2 Instance Connect | ブラウザからEC2へ接続する方法 |
| セッションマネージャー | Systems Managerを使った接続方法 |
| SSHクライアント | ローカルPCのTerminalからSSH接続する方法 |
| EC2シリアルコンソール | トラブルシューティング用の接続方法 |

今回は **EC2 Instance Connect** を使用する。

---

## 4. ユーザー名を確認する

接続画面では、ユーザー名が表示される。

Amazon Linux系のEC2では、通常以下のユーザー名を使う。

```bash
ec2-user
```

ユーザー名が `ec2-user` になっていることを確認する。

---

## 5. 「接続」をクリックする

内容を確認したら、画面右下の **接続** ボタンをクリックする。

すると、ブラウザ上に黒いターミナル画面が開き、EC2インスタンスにログインできる。

---

## 接続成功の確認

接続に成功すると、以下のような表示が出る。

```bash
Amazon Linux 2 AMI
```

また、プロンプトが以下のように表示される。

```bash
[ec2-user@ip-xxx-xxx-xxx-xxx ~]$
```

この表示になれば、EC2インスタンスへ接続できている。

---

## MacのTerminal接続との違い

| 接続方法 | 特徴 |
|---|---|
| Mac TerminalからSSH接続 | `.pem` ファイルや `ssh -i` コマンドを使う |
| EC2 Instance Connect | AWSコンソールからブラウザ上で接続できる |

EC2 Instance Connectでは、Terminalを開いたり、手動でSSHコマンドを入力したりしなくても接続できる。

---

## 今日のポイント

- EC2インスタンスは `実行中` の状態である必要がある。
- EC2一覧画面で接続したいインスタンスを選択する。
- 画面上部の **接続** ボタンから接続画面を開く。
- 今回は **EC2 Instance Connect** を使って接続した。
- ユーザー名は通常 `ec2-user` を使用する。
- 接続に成功すると、ブラウザ上でLinuxのコマンド操作ができる。
- `[ec2-user@...]$` の表示が出れば、EC2へログインできている。

---

## 自分用メモ

EC2 Instance Connectで接続する流れは以下。

1. EC2インスタンス一覧を開く
2. 接続したいインスタンスを選択する
3. **接続** ボタンをクリックする
4. **EC2 Instance Connect** を選択する
5. ユーザー名が `ec2-user` になっていることを確認する
6. **接続** をクリックする
7. ブラウザ上のターミナルでEC2を操作する

EC2 Instance Connectは、初心者でもAWSコンソールから簡単にEC2へ接続できる便利な方法。
