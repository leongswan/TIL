# TIL（2026/2/19）C言語：基本文法 / 実行順序 / main関数 / printf

## 1. C言語の基本構成
- Cプログラムは基本的に次の流れで書く  
  **(1) ヘッダ（ライブラリ）を読み込む → (2) main関数から実行 → (3) 必要な処理 → (4) 終了**
- 実行は **main関数から開始**される（ここが入口）。

---

## 2. 重要キーワード
### `#include <stdio.h>`
- 標準入出力のライブラリ（ヘッダ）を読み込む。
- `printf` などの入出力関数を使えるようにする。

### `int main(void)`
- プログラムの開始地点（エントリーポイント）。
- `int` は「終了ステータス（戻り値）」を返すという意味。
- `void` は「引数を受け取らない」という意味。

### `printf("Hello\n");`
- 文字を画面に表示する関数。
- `\n` は改行（バックスラッシュ + n）。

### `return 0;`
- `main` を終了し、OSへ **正常終了（0）** を返す。
- 0以外は「異常終了」を表すことが多い。

---

## 3. 実行順序（このコードの場合）
1. `main` が呼ばれる  
2. `printf` が実行されて `Hello` が表示される（改行あり）  
3. `return 0;` で終了

---

## 4. サンプルコード（基本形）
```c
#include <stdio.h>  // ライブラリ

int main(void) {    // main関数（ここから実行）
    printf("Hello\n");  // printf関数（表示）
    return 0;       // 正常終了
}
```

---

## 5. 今日のポイント
- Cはmain関数から実行される
- printf を使うには #include <stdio.h> が必要
- 改行は \n
- return 0; は「正常終了」を意味する
