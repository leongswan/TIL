# TIL（2026/02/12）— Pythonの条件分岐（if / elif / else）

## 今日の学び（要点）
- **条件分岐**：条件によって処理内容を変える仕組み
- Pythonでは **if文** を使う
- 条件式は **比較演算子** と **論理演算子** を組み合わせて書ける
- Pythonは **改行とインデント（空白）に意味がある**（超重要）
- 分岐は基本的に `if` → `elif` → `else` の順で評価され、最初に真になったものだけ実行される

---

## if文の基本構文
```python
if 条件:
    条件がTrueのときの処理
elif 別の条件:
    別の条件がTrueのときの処理
else:
    どれもTrueでないときの処理
比較演算子（大小・一致の判定）
==：等しい

!=：等しくない

>：より大きい

<：より小さい

>=：以上

<=：以下

例：

x = 10
print(x >= 5)  # True

```

---

## 論理演算子（条件の組み合わせ）
- and：両方TrueならTrue

- or：どちらかTrueならTrue

- not：True/Falseを反転

```python
例：

age = 20
has_ticket = True

if age >= 18 and has_ticket:
    print("入場できます")
```

---

## 記法の注意（インデントが超重要）
- : の後は 必ず改行して、インデント（通常スペース4つ）

- インデントがずれると、エラー or 意図しない動きになる

✅ 正しい例：
```python
score = 85
if score >= 80:
    print("合格")
```
❌ よくあるミス（インデントなし）：
```python
if score >= 80:
print("合格")  # IndentationError
```

---

## 例：点数で評価を分ける（if / elif / else）

```python
score = 72

if score >= 80:
    print("A")
elif score >= 60:
    print("B")
else:
    print("C")
```
---

## ポイント
- より厳しい条件（高い点数）から先に書くと安全

- elif score >= 60 は、上の if がFalseだった場合だけ判定される
