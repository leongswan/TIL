# TIL（2026/02/16）— Pythonの繰り返し処理・range()・関数・組み込み関数（abs）

## 今日学んだこと（要点）
- **繰り返し処理**には主に **for文** と **while文** がある  
- **range()** は連番（整数の列）を作るためによく使う
- **関数**は「特定の処理をまとめたもの」
  - **組み込み関数**：Pythonに最初から用意されている（例：`abs()`）
  - **自作関数**：必要に応じて自分で定義する（`def` で作る）

---

## 1) 繰り返し処理

### for文（回数や要素数が決まっているときに便利）
```python
for i in range(5):
    print(i)  # 0,1,2,3,4
```

## while文（条件が満たされている間ずっと繰り返す）
```python
x = 0
while x < 5:
    print(x)
    x += 1
```

---

## 2) range() の基本
```python
range(start, stop, step)
```

### ルール
- デフォルトは 0 から開始
- デフォルトの増分は +1
- stop の直前で止まる（stopは含まれない）

### 例
```python
range(5)        # 0,1,2,3,4
range(2, 6)     # 2,3,4,5
range(1, 10, 2) # 1,3,5,7,9
```

- 何度も使う処理や、まとまりとして管理したい処理を 関数にすると便利

### 組み込み関数の例（abs）

## 4) 演習：abs() を使って絶対値を出力する
### 問題：すでに定義されている test の絶対値を出力する

## 今日のまとめ
- 繰り返し：for は回数/要素で回す、while は条件で回す
- range() は 0開始・+1増加・stopは含まない（必要なら start/step も指定）
- abs() は組み込み関数で、絶対値を簡単に求められる
