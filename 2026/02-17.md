# TIL（2026/02/17）— Pythonの関数（定義・仮引数/実引数）と平均を返す関数avg()

## 今日学んだこと（要点）
- **関数**は「特定の処理をまとめて、何度でも使えるようにする仕組み」
- 関数を使うと、処理の再利用・整理・読みやすさが上がる
- **仮引数**：関数を定義するときに受け取る引数（例：`a, b, c`）
- **実引数**：関数を呼び出すときに渡す値（例：`4, 2, 6`）

---

## 1) 関数の基本形

### 定義（def / 仮引数 / return）
```python
def 関数名(仮引数):
    return 返り値
```
例：
```python
def add(a, b):
    return a + b
```
### 呼び出し（実引数）
```python
関数名(実引数)
```
例：
```python
add(4, 5)  # 実引数 4,5 を渡して実行
```

---

## 2) 演習：3つの整数の平均を返す関数 avg(a, b, c)
### 目的
- 与えられた3つの整数値の平均を返す avg(a, b, c) を作る
- avg(4, 2, 6) を実行して結果を出力する

### 解答コード
```python
def avg(a, b, c):
    return (a + b + c) / 3

# 4,2,6を引数としてavg()を実行し、その結果を出力してください。
print(avg(4, 2, 6))
```

---

## 今日のまとめ
- def で関数を定義し、return で値を返す
- 仮引数は受け取る箱（a,b,c）、実引数は渡す値（4,2,6）
- 平均は (a+b+c)/3 で求められる
