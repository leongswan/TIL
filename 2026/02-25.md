# TIL（2026/2/25）C言語：ビット演算（AND/OR）とシフト演算（<< / >>）

## 1. ビット演算とは
- 整数を **2進数（ビット列）** として扱い、ビット単位で計算する演算。
- 代表例：`&`（AND）, `|`（OR）, `^`（XOR）, `~`（NOT）

---

## 2. AND（&）と OR（|）
### AND（&）
- **両方が1のときだけ1**、それ以外は0

例：
```text
  00011100
& 00010010
----------
  00010000
```
### OR（|）
- **どちらかが1なら1**、両方0なら0

例：
```text
  00011100
| 00010010
----------
  00011110
```

---

## 3. シフト演算（<< / >>）
**左シフト** `<<`

- ビットを左にずらす（空いた右側は0で埋める）
- **だいたい「2倍」方向**（`x << n` は `x × 2^n`）

例：
```text
00011100 << 2  → 01110000
```
**右シフト** `>>`

- ビットを右にずらす（空いた左側は基本0で埋まる※）
- **だいたい「1/2」方向**（`x >> n` は `x ÷ 2^n` のイメージ）

例：
```text
00011100 >> 2  → 00000111
```

※符号付き整数（負数）の場合、処理系により左側が1で埋まることがある（算術シフト）。

---

## 4. コード例（AND / 左シフト）
### 例の値

- **a = 0x10** → `00010000`
- **b = 0x11** → `00010001`

ANDすると：
```text
00010000
&00010001
---------
00010000  (= 0x10)
```

### サンプルコード（修正版コメント付き）
```c
#include <stdio.h>

int main(void){
    int a = 0x10;   // 00010000
    int b = 0x11;   // 00010001

    int ans = a & b; // & = AND, | = OR

    printf("%x\n", ans);     // 10
    printf("%x\n", b << 2);  // 左シフト（b * 4） -> 44

    return 0;
}
```

---

## 5. 注意点（今回のメモで混ざりやすい所）

- `<<` は **左シフト**
- `>>` は **右シフト**

---

## 今日のまとめ

- `&`（AND）：両方1のビットだけ1になる
- `|`（OR）：どちらか1なら1になる
- `<<`（左シフト）：値を `×2^n` 方向に動かす
- `>>`（右シフト）：値を `÷2^n` 方向に動かす（符号付きは注意）
