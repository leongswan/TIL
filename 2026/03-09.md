C言語の再帰処理を使って、フラクタル図形を描画する問題を学習した。

## 問題の概要
`pat` という2次元配列を生成パターンとして使い、  
深さ `d` のフラクタル図形を `*` と空白で表示するプログラムを読む問題だった。

生成パターンは次の通り。

~~~c
int pat[2][2] = {
    {1, 1},
    {1, 0}
};
~~~

- `1` は `*`
- `0` は空白
- 深さが1増えるごとに、1つ小さい図形の各文字を `pat` に従って置き換える

---

## この問題で理解したこと

### 1. 2次元配列の行数・列数の求め方
~~~c
int p_rn = sizeof pat / sizeof pat[0];        // 行数
int p_cn = sizeof pat[0] / sizeof pat[0][0];  // 列数
~~~

- `p_rn` は行数
- `p_cn` は列数

---

### 2. 最終的な図形サイズの求め方
深さ `d` の図形サイズは、生成パターンの行数・列数を `d` 回掛ける。

~~~c
rn = cn = 1;
for (i = 0; i < d; i++) {
    rn *= p_rn;
    cn *= p_cn;
}
~~~

---

### 3. 再帰処理の考え方
`exists_at(i, j, d)` は、  
「深さ `d` の図形の i 行 j 列に `*` があるか」を判定する関数。

考え方は次の通り。

- 深さ0なら必ず `*`
- 1つ上の階層が空白なら、今の位置も空白
- そうでなければ、生成パターン `pat` の対応位置を見る

~~~c
if (d == 0) {
    return 1;
} else if (exists_at(i / p_rn, j / p_cn, d - 1) == 0) {
    return 0;
} else {
    return pat[i % p_rn][j % p_cn];
}
~~~

---

### 4. 商と余りの使い分け
この問題の重要ポイント。

- `i / p_rn`, `j / p_cn`
  - 1つ上の階層のどのマスに属するかを見る
- `i % p_rn`, `j % p_cn`
  - 現在の生成パターン内のどの位置かを見る

---

### 5. 三項演算子の使い方
~~~c
putchar(exists_at(i, j, d) ? '*' : ' ');
~~~

- 条件が真なら `*`
- 偽なら空白

## 修正版の完成コード
※貼っていたコードの誤記を直した版  
（`for` 文の条件、`j / p_cn`、空白文字 `' '` などを修正）

~~~c
#include <stdio.h>

/*
  * = 1
    = 0
*/

int pat[2][2] = {
    {1, 1},
    {1, 0}
};

int p_rn = sizeof pat / sizeof pat[0];         // 行数
int p_cn = sizeof pat[0] / sizeof pat[0][0];   // 列数

void print_frac(int);
int exists_at(int, int, int);

/* フラクタル図形を描画する関数
   d: 深さ
*/
void print_frac(int d) {
    int i, j, rn, cn;
    rn = cn = 1;

    for (i = 0; i < d; i++) {   // 最終図形の縦横サイズを求める
        rn *= p_rn;
        cn *= p_cn;
    }

    for (i = 0; i < rn; i++) {
        for (j = 0; j < cn; j++) {
            putchar(exists_at(i, j, d) ? '*' : ' ');//3項演算子(条件式)?真：偽
        }
        putchar('\n');
    }
}

/* 深さdのフラクタル図形の i行j列目が
   空白か '*' かを判定する
   0: 空白
   1: '*'
*/
int exists_at(int i, int j, int d) {
    if (d == 0) {
        return 1;
    } else if (exists_at(i / p_rn, j / p_cn, d - 1) == 0) {//一つ小さい深さでのexists_at
        return 0;
    } else {
        return pat[i % p_rn][j % p_cn];
    }
}
~~~
