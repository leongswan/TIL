# TIL（2026/2/27）C言語：ポインタ（pointer）

## 1. ポインタとは
- **変数そのものの値ではなく、「変数が置かれているメモリ上の住所（アドレス）」を扱う仕組み**
- 「どこにあるデータか」を指し示す（参照する）ために使う

---

## 2. ポインタ型（例：int*）
- `int*` は **int型のデータが置かれているアドレス** を入れるための型
- つまり `int* a;` は「int型を指すポインタ変数 a」

---

## 3. &（アドレス演算子）
- `&x` は **変数xのアドレス** を取り出す
- 例：`int* a = &x;` は「aにxの住所を入れる」

---

## 4. *（間接参照 / デリファレンス）
- `*a` は **aが指している先の値（中身）** を取り出す
- 例：`*a` は `x` と同じ値になる（aがxを指している場合）

---

## 5. 例：アドレスを表示する
```c
#include <stdio.h>

int main(void){
    int x = 6;
    int* a = &x;

    printf("%p\n", (void*)a);   // xのアドレス（ポインタの中身）を表示
    return 0;
}
```
### ポイント

- `%p` は アドレス表示専用
- `printf("%p", a);` でも動く環境は多いけど、Cでは安全のため *(void) にキャスト**するのが定番

---

## 6. 例：値（中身）を表示する

```c
#include <stdio.h>

int main(void){
    int x = 6;
    int* a = &x;

    printf("x  = %d\n", x);     // 6
    printf("*a = %d\n", *a);    // 6（aが指す先の値）
    return 0;
}
```

---

## 今日のまとめ

- **ポインタ**：メモリ上のアドレスを入れる変数
- `&`：変数のアドレスを取る
- `*`：ポインタが指す先の値を取り出す（デリファレンス）
- `%p`：アドレス表示（推奨：`(void*)` にキャスト）
