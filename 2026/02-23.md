# TIL（2026/2/23）C言語：計算と演算 / 算術演算子・論理演算子 / doubleとfloat

## 1. 変数の準備（例）
```c
int main(void){
    int a = 0;
    int b = 4;
    int c = 100;
    double d = 3.2;
    float f = 4.5f;  // floatの定数は f を付けるのが基本
}
```

---

## 2. 算術演算子（計算に使う）

- `+` 加算
- `-` 減算
- `*` 乗算
- `/` 除算
- `%` 剰余（あまり）※整数のみ
- `++` / `--` 1増やす / 1減らす

### 例：足し算

```c
int ans;
ans = a + b;
printf("%d\n", ans);  // 0 + 4 = 4
```

### 注意：整数の割り算

- int / int は 小数が切り捨てされる

```c
int x = 5 / 2;     // 2 になる（2.5ではない）
double y = 5 / 2;  // これも 2.0（計算がint同士だから）
double z = 5.0 / 2; // 2.5（どちらかをdoubleにする）
```

---

## 3. 論理演算子・比較演算子（条件判定に使う）
### 比較演算子（結果は真=1 / 偽=0）

- `==` 等しい
- `!=` 等しくない
- `<` `<=` `>` `>=`

#### 論理演算子（条件を組み合わせる）

- `&&` AND（両方真なら真）
- `||` OR（どちらか真なら真）
- `!` NOT（反転）

### 例：比較（a==b）
```c
int ans;
ans = (a == b);      // 0==4 なので偽 → 0
printf("%d\n", ans); // 0
```
### 例：条件の組み合わせ
```c
int ok = (b > 0 && c >= 100);  // (4>0)かつ(100>=100) → 1
printf("%d\n", ok);            // 1
```

---

## 4. double と float の違い（小数の型）

- float：単精度（32bit）
- double：倍精度（64bit）
- doubleの方が より精密（誤差が小さい）で、Cでは小数計算は doubleが基本になりやすい。

### printfでの表示

- double / float は printf ではどちらも %f を使う（floatは渡すときにdoubleに拡張される）
```c
printf("%f\n", d); // double
printf("%f\n", f); // float
```
### floatのリテラルは f を付ける
```c
float f1 = 4.5f;   // OK（float）
double d1 = 4.5;   // OK（double）
```

## 今日のまとめ

- 算術演算子は計算（+ - * / %）に使う
- 比較/論理演算子は条件判定（== != && ||）に使い、結果は 1 or 0
- 整数同士の割り算は小数が切り捨て
- doubleはfloatより高精度。小数計算はdoubleが基本になりやすい
