# TIL（2026/2/23）C言語：条件分岐（if）/ 繰り返し（while・for）

## 1. 条件分岐（if / else if / else）
- 条件によって処理を分ける構文。
- 上から順に条件を判定し、**最初に真になったブロックだけ実行**される。
- どの条件にも当てはまらない場合は `else` が実行される。

### 例（a=15 の場合）
```c
if (a < 10) {
    printf("under 10.");
}
else if (a < 15) {
    // aが10以上15未満のときに実行される
}
else {
    printf("upper 10.");
}
```
### この例のポイント

- `a = 15` のとき

  - `a < 10` は偽

  - `a < 15` も偽（15は「未満」ではない）

  - よって `else` が実行され、`upper 10.` が表示される
  
---

## 2. 繰り返し（while文）

- **条件が真の間**、同じ処理を繰り返す。
- ループ内で条件が変わらないと **無限ループ**になるので注意。

```c
int i = 0;
while (i < 100) {
    printf("hello\n");
    i = i + 1;  // iを増やして終了条件へ近づける
}
```
### ポイント

- `i`は 0〜99 の100回ループする（合計100回`hello` を出力）

---

## 3. 繰り返し（for文）

- ループ回数が決まっているときに書きやすい。
- `for(初期化; 条件; 更新)` の形でまとめて書ける。

```c
for (int i = 0; i < 100; i++) {
    printf("hello\n");
}
```

### ポイント

- whileの例と同じく 100回繰り返す
- for文は「初期化」「条件」「更新」が1行にまとまっていて見通しが良い

---

## 4. まとめ（使い分け）

- **if / else if / else**：条件に応じて処理を分岐する
- **while**：条件が成立する間ずっと繰り返す（回数が未確定でも使える）
- **for**：回数が決まっている繰り返しに向いている

---

## サンプル全体（整理版）

```c
#include <stdio.h>

int main(void){
    int a = 15;

    if (a < 10) {
        printf("under 10.\n");
    } else if (a < 15) {
        printf("under 15.\n");
    } else {
        printf("upper 10.\n");
    }

    int i = 0;
    while (i < 100) {
        printf("hello\n");
        i = i + 1;
    }

    for (int j = 0; j < 100; j++) {
        printf("hello\n");
    }

    return 0;
}
```
