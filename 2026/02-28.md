# TIL（2026/2/28）C言語：再帰（再帰関数）と sum(n)

## 1. 再帰的プログラム（再帰関数）とは
- **関数の中で、自分自身を呼び出す**ことで処理を進めるプログラムのこと
- 重要なのは次の2つ
  - **終了条件（ベースケース）**：いつ再帰を止めるか
  - **再帰ステップ**：問題を小さくして自分自身を呼ぶ（例：n → n-1）

---

## 2. sum(n)：0〜nまでの合計を返す考え方
### イメージ（例：sum(3)）
- `sum(3) = 3 + sum(2)`
- `sum(2) = 2 + sum(1)`
- `sum(1) = 1 + sum(0)`
- `sum(0) = 0`（ここが終了条件）

なので  
`sum(3) = 3 + 2 + 1 + 0 = 6`

---

## 3. C言語での実装例（再帰）
```c
#include <stdio.h>

int sum(int n);

int main(void){
    int ans = sum(10);
    printf("%d\n", ans);  // 55
    return 0;
}

int sum(int n){
    if (n <= 0) {
        return 0;         // 0以下は 0 を返して終了
    }
    return n + sum(n - 1); // n を足して、1つ小さい問題へ
}
```
### 実行結果

- `sum(10) = 10 + 9 + ... + 1 + 0 = 55`

---

## 4. ポイント（落とし穴）

- `if (n <= 0) return n;` でも動くことはあるけど、
**nが負のときに負の値が混ざる**可能性があるので、合計目的なら `return 0;` が安全
- 再帰は分かりやすい反面、呼び出し回数が増えると **スタックを消費**する（大きいnでは注意）

---

## 今日のまとめ

- 再帰は「自分自身を呼び出す」処理
- 必ず **終了条件** が必要
- `sum(n)` は n + `sum(n-1)` で問題を小さくして解ける
- `sum(10)` の結果は **55**
